# Foodies is an online cafe for ordering dishes from a coffee shop.

## Characteristics.

• * Splash Screen
• Sort products by category.
• Search for products and highlight the entered letters in the product name
• Filtration of products

### Project Configuration

- compileSdk = 34
- minsk = 24
- targetSdk = 34
- source Compatibility = Java Version.VERSION_1_8
- target Compatibility = Java Version.VERSION_1_8
- jvm Target = "1.8"

### Technologies and tools

- Kotlin 1.5.3
- MVVM
- Jetpack Compose
- Hilt
- Retrofit
- OkHttp
- Flow
- Lottie

## How to install?

You can install this app for testing on your Android smartphone. To download the signed application, go to your Chrome browser and copy-paste the download link:
https://drive.google.com/file/d/1rlRitMXCcv4SfzOpFMgv_DxItQXy1YeU/view?usp=drive_link
